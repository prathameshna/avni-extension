# Avni Browser Extension

Avni is an AI productivity and workflow management extension designed for seamless integration across popular AI platforms.

## Supported Platforms
- Claude (claude.ai)
- ChatGPT (chatgpt.com)
- Gemini (gemini.google.com)
- Grok (grok.com)
- DeepSeek (deepseek.com)
- Meta AI (meta.ai)

## Features
- Smart Clipboard
- Context Management
- Token Tracking & Management
- Unified Dashboard
